Understand your users. Instantly.

WebGaze turns any webcam into a gaze tracker, giving researchers real-time visibility into where users look — without expensive hardware. Built for fast, iterative UX research.

Feature

Real-time Heatmaps

Visualize attention density across your UI as users interact with it.

Feature

Fixation Detection

Automatically identify when a user is focused vs. scanning rapidly.

Feature

AOI Analytics

Define regions of interest and get dwell-time stats automatically.

Research shows users form first impressions within 50ms of landing on a page. With WebGaze, you can measure those critical moments and optimize accordingly. Works in any modern browser — no plugins required.

Accuracy

~50–100px Error

Sufficient for button-level and section-level analysis in most layouts.

Performance

30fps Processing

Runs on-device via WebAssembly — no server round-trips.

Privacy

100% Client-side

No video data ever leaves the browser. Only coordinates are recorded.
